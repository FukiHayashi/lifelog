# 生活記録アプリ
## 概要

復職するにあたり、日々の生活記録を取る必要がありました。 手書き、Excelなど取る方法は何でもOKでしたが、提出するフォーマットに落とし込むことが手間でした。
本アプリはこの手間をなくし、生活記録を入力するだけで下図の様に可視化まで行うことを目的に開発しました。
